Icon System
Techne uses inline SVG icons throughout — no emoji, no icon font, no external icon library required. Icons are rendered via a shared Icon component that accepts an SVG d path string and a size prop. All icons use a 16×16 or 24×24 viewBox, 1.5px stroke, round linecap/linejoin. Color inherits from currentColor.
// Icon component (HUDLayout, intranet-src)
function Icon({ d, size = 16 }) {
return <svg width={size} height={size} viewBox="0 0 16 16"
fill="none" stroke="currentColor" strokeWidth="1.5"
strokeLinecap="round" strokeLinejoin="round">
<path d={d} />
</svg>
}
